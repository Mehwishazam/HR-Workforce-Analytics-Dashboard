# HR-Workforce-Analytics-Dashboard
HR analytics dashboard focused on workforce metrics and attrition insights.
Provides a complete overview of the workforce using key HR metrics

Analyzes total employees, current employees, attrition, attrition rate, and average employee age

Highlights workforce distribution by department, gender, age groups, and education fields

Examines attrition patterns to identify employee turnover trends

Includes job satisfaction insights to understand potential retention factors

Supports data driven HR decision making and workforce planning
